Enomy Finances

Enomy Finances is a website that I created for mortgages, savings, money conversion and real time investments for users.

Technologies




Features

Website features.

Register and Login that allows users to create an account and use the website handled by Spring Security for higher security.

     

Database used for storing user accounts that are used for log in and register on the website.

  

Currency Conversion functionality that allows users to perform currency conversion based on the currency available on the website in real time.

  

Savings and Investments functionality that allows users to calculate investment expectations based on plans to make savings.

     

Users management that can be performed by the admin of the website to view, delete users on the website.
