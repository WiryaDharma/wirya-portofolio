Know Your Neighborhood

Know Your Neighborhood is a website that I created to allow users to find stores around the users neighborhood.

Technologies




Features

Website features.

Database used to store all data on the website and support each functionality.

Add Store functionality that allows users to add and post stores on the website.

View Store functionality that allows users to view stores on the website and perform actions such as update, delete for stores.

Update Store functionality that allows users to update the stores information on the website.
