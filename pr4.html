ABC Cars Portal

ABC Cars Portal is a website that I created in order to allow users to buy and sell used cars on the website.

Technologies




Features

Website features.

Register and Login that allows users to create an account and use the website handled by Spring Security for higher security.

  

The database used to store all data on the website to support each functionality.

  

Profile functionality that allows users to view their profile information.

  

Update Profile functionality that allows users to update their profile information.

  

View or Search Cars functionality that allows users to search and view cars on the website.

  

Car Details functionality that allows users to see the details of the cars on the website.

  

Bidding Price functionality that allows users to bid for cars on the website.

  

Post Cars functionality that allows users to upload and post cars on the website.

  

Cars Management functionality that allows admins to manage cars such as update, delete cars.

  

Users Management functionality that allows admins to manage the users such as view, deactive
